üìä Redis Settings (Shadow Mode Data Storage)

Complete Redis connection string including credentials and database

Redis URL Format:

redis://[username:password@]host:port[/database]

Examples:

Local Redis:
redis://localhost:6379

Redis with password:
redis://:mypassword@localhost:6379

Redis Cloud (username + password):
redis://username:password@redis-12345.cloud.redislabs.com:12345

Redis Labs with specific database:
redis://default:mypassword@redis-19249.crce163.us-east-1-mz.ec2.qa-cloud.redislabs.com:19249/0

URL Components:

  • username: Redis username (often "default" for Redis Cloud)
  • password: Your Redis password/auth token
  • host: Redis server hostname or IP
  • port: Redis port (default: 6379)
  • database: Database number (optional, default: 0)
